## ANI	

+ nestjs 
+ swagger
+ jwt auth

## Installation

```bash
$ npm install or yarn
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run build 
$ npm run start:prod
```

## Running the app with pm2

```bash
npm i pm2 -g
pm2 start ./dist/main.js
```

# Docker

```
dock build -t app .
```
